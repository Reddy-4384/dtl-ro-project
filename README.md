#DTL-RO Project  
Dynamic Transport Logistics & Route Optimization Tool  

## Description  
This project optimizes delivery routes for vehicles using Python.  
It considers distance, traffic, and dynamic order updates to find the best route.

## Features  
- Route optimization (Greedy + 2-opt)  
- Dynamic order insertion  
- Traffic simulation  
- CSV output generation  
- Graph visualization (Matplotlib)  
- Map visualization (Folium)  

## Technologies Used  
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Folium  

## How to Run  
```bash
python main.py
